ABOUT HTML ENTITES

Reserved characters in HTML must be replaced with character entities.

Some Characters are reserved in HTML.

If we use the less than(<)or greater than (>) signs in our text, then the browser might mix them with tags.

A character entity looks like this:
&entity name;
OR
&#entity_number;
